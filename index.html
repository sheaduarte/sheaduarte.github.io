About Me

I'm a PhD Candidate in Joy Geng’s lab at the UC Davis Center for Mind & Brain.

My doctoral research investigates how visual & multisensory perception interact with memory to guide naturalistic behavior. To study these mechanisms, I use psychophyics and virtual reality (VR) with integrated eyetracking.

When I'm not in the lab, I spend my time in the ceramics studio, cooking and baking, and traveling as much as possible.

Multisensory Influences on Memory

Multimodal information is ubiquitous in natural environments and has been shown to influence cognition from perception to learning, and neural processing, from neuronal firing to coherence of large-scale brain networks. However, most research in human memory has used only visual stimuli, so our understanding of how non-visual and multimodal information influences memory is limited. We (Duarte et al., 2022) found that encoding a visual object along with its characteristic sound (e.g., a dog & a bark) led to improved recollection-based recognition memory later on. Recollection is a form of episodic memory that supports memory of items on the basis of memory for details of the surrounding context in which that item was situated. This is in contrast to the behaviorally & neurally distinct process of familiarity, which supports memory for individual items. Our findings imply that encoding an object in a multisensory manner fundamentally changes the way in which that information is incorporated into long-term memory, and that perceiving audiovisual items leads to overall stronger episodic memory rather than just better memory for the multimodal item itself.
